Keyboard Shortcuts


These are all the keyboard shortcuts available in WaveTracker.

General

Key Action
Alt+F4 Exit
Ctrl+N Create a new file
Ctrl+O Open file
Ctrl+S Save file
Ctrl+E Open export dialog
Return Play / stop
Alt+Return Play from cursor
Ctrl+Return Play row
F5 Play from start
F7 Play from cursor
F8 Stop
F2 - F9 Select octave 0 - 7, if no other keys are configured.
Alt+F9 Mute/Unmute channel
Alt+F10 Solo channel
F12 Kill sound and reload sound configuration

Frame Editor

Key Action
Ctrl+X Cut
Ctrl+C Copy
Ctrl+V Paste
Ctrl+B Paste and overwrite
Ctrl+A Select all
Ctrl+Z Undo last action
Ctrl+Y Redo last action
Esc Deselect
Del Delete selection / selected row
Home Move to top of pattern
End Move to end of pattern
Space Toggle editing mode
Backspace Delete the row above
Ctrl+G Interpolate selection
Alt+R Humanize, slightly randomizes volume
Shift+Alt+Scroll Scale volume
Ctrl+R Reverse selection
Ctrl+F1 Transpose down one semitone
Ctrl+F2 Transpose up one semitone
Ctrl+F3 Transpose down one octave
Ctrl+F4 Transpose up one octave
Shift+F1 Decrease pattern values
Shift+F2 Increase pattern values
Shift+F3 Decrease pattern values by 10
Shift+F4 Increase pattern values by 10
Ctrl+Right Go to next frame
Ctrl+Left Go to previous frame
Ctrl+D Duplicate frame
Tab / Alt+Left / Alt+Scroll Move to next channel
Shift+Tab / Alt+Right / Alt+Scroll Move to previous channel
Numpad 0-9 Select instrument 0 - 9
Alt+Numpad 0-9 Set step size, 0 - 9
Numpad / Decrease octave
Numpad * Increase octave
Ctrl+Up Next instrument
Ctrl+Down Previous instrument
